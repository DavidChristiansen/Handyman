# ToDo Plugin - UNFINISHED
ToDo lists.

##### Alias: todo
##### Help: help todo

## Commands
- ```list```: shutdown pc
- ```add```: restart pc
- ```delete```: hibernate pc
- ```done```: log off

## Usage
#### list
```cs
todo
// or
todo list
```
#### add
```cs
todo add n:{Name} d:{Description}
// example
todo add n:Name d:Description of todo item
```
#### delete
```cs
todo delete {ItemName}
```
#### done
Mark ToDo item done.
```cs
todo done {ItemName}
```

## Info
You can find saved data in **Documents** folder under name **{username}.todoHandyman**
