# REST Plugin
Make GET, POST requests from launcher.

##### Alias: rest
##### Help: help rest

## Commands
- ```get```: make GET request.

## Usage
Make GET Request - **UNFINISHED**
```cs
rest get {url} var1 var2 var3 var4 ... varx
```
Make POST Request - **UNFINISHED**
```cs
rest post {url} arg1 arg2 arg3 ... argx
```
